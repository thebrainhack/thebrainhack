真正的比特币BSV是一个创造价值的系统？

为什么BTC是一个吸收价值的系统，而真正的比特币BSV是一个创造价值的系统？

甚至在你进入更深层次的技术探究之前，从他们截然不同的、显而易见的外在举止中可以看出，比特币核心（BTC）是一个吸收价值的系统，而比特币中本聪愿景（BSV）是一个创造价值的系统。也许只要在BTC（在迈阿密）和BSV（在苏黎世）最近举行的两个会议的气氛中体验几分钟，你就会清楚地知道区别在哪里：一个是所有关于激光眼的HODLing尖叫的兴奋，而另一个主要是平静和专业的商业演讲。

在一个更实质性的问题上，一个系统是一个创造价值的系统还是一个吸收价值的系统，与它的经济价值结构有关。关于更详细的理论讨论，见。BTC和BSV，真正的区别是什么？)

BSV是一种贡献经济，而BTC是一种提取经济。贡献经济激励人们参与生产经济，创造新的价值，这些价值本身是真实的，而不是其他人的纯粹投机，而提取经济则引诱人们进入其中，轻松提取现有价值。 贡献经济增加了共同利益，使贡献者自己和后来的人都受益，但提取经济则奖励少数人，他们通过人为地创造机会主义的时间差异来利用其他人的利益。

以此类推，BSV被设计成可以建立经济的 “土地”，而BTC是人们可以持有的 “黄金”。 土地创造价值，而黄金吸收价值。

此外，如果BTC真的成为估值达数万亿美元的数字黄金，这种独特的HODLing历史将在人们之间造成极其不平衡的财富分配，集中在对生产力没有贡献也没有让其他人生活得更好的一小部分人身上。 因此，这不仅仅是价值吸收，而且是极其不公平的价值吸收。 相比之下，如果BSV在类似数字土地的水平上取得成功，就不会出现同样的问题。

考虑一下像美国这样的国家的整个经济。今天，纯粹的土地交易和土地价值只构成了美国经济的一小部分。 但一切都建立在这片土地上。房地产市场总价值约为50万亿美元（包括商业和住宅），而土地价值只是其中的一小部分，大约是10万亿美元。 同时，美国每年的GDP约为20万亿美元，而美国的总财富则远远超过100万亿美元。

通过上述数字，请问这个问题：今天人们是否抱怨这片土地上最早的定居者得到了不公平的财富，或者说太富有了，仅仅是因为他们来得早，得到了土地？对早期的土地所有者来说，显然是有好处的。但它并没有对经济构成难以承受的负担，无论是心理上还是经济上。

这有两个原因。

首先，即使今天的土地价值比仅仅50年前高得多，比早年也高得多，但它只是经济总量的一小部分。

第二，也许更重要的是，几百年来，美国的经济发展一直是投机和建设的相当健康的组合，总体上强调后者，即实际建设。正是建筑部分加深和加速了土地价值的交换。

也就是说，早期的土地拥有者一般不会成为土地棚户区居民。 他们出售了土地。而买家，虽然有相当数量的投机者，但大多数是为了开发而购买土地。

其结果是经济每年产生超过20万亿美元的价值和超过100万亿美元的财富储蓄，所有这些都建立在价格低于10万亿美元的土地上，其中大部分在最近支付当前市场价格的买家手中。

简而言之，我们可以说美国经济总体上被定性为一个价值创造系统，而不是一个价值吸收系统。美国的大部分财富来自人们的劳动和创造，而不是来自土地所有权。

这就是一个健康的加密货币生态系统应该建立的经济类型。

而这正是根据中本聪设想的比特币（BSV）正在做的事情。

数十家应用/解决方案公司已经在BSV领域建立，还有数百家正在产生。尽管BTC Core在万事达卡和硅谷某些势力的暗中推动下，组织了压倒性的打击士气的造谣运动，但已经有超过1000名开发者在这个领域。

如果BTC有同样的比例，它现在在自己的空间里会有超过20万名开发者（因为到今天为止它的市场估值是BSV的200倍以上）。

但现实却恰恰相反。 除了极少数的例外，如闪电网络（由于根本性的缺陷而无法运作），BTC上几乎没有发生开发活动。在BTC团队中，总共只有不到8个核心开发人员。 这不仅是因为BTC投机市场的动态对开发者的心理产生了破坏性的影响（当硬币让人致富而不需要制造实际的产品时，谁还愿意制造产品呢），而且还因为BTC被设计成一个HODLING（持有）系统，以培养HODLING文化，而不是生产价值的工作文化。

对于一个应该基于工作证明原则的系统来说，这是多大的讽刺啊!

“数字黄金 “的说法说明了一切。 与生产技术不同，黄金不创造或生产东西。 沃伦-巴菲特说得很对，他说：”黄金是一种宠物石。” 如果说这句话对黄金来说是不公平的，那么对BTC来说就准确多了。

投机币的价值是BTC的整个经济。 它没有一个设计，没有一个计划，甚至没有一个模糊的希望来创造一个基于人类创造力的多层次和多方面的经济。

有一些人认为他们也可能通过持有BSV币而发财，但这种投机部分远远小于BTC的投机部分，无论如何，适度的投机可能是一个经济体组合中的好成分。

而且，BSV的价格被市场进一步严重压制，从长期前景来看，这对 “数字土地 “投资者来说是件好事，因为这意味着土地在出售。

事实上，BSV价格没有暴涨，对发展来说也是一件健康的事情。 它冲走了过度的投机，并对剩余的社区，特别是开发商和投资者产生了硬化作用。这是一种必要的心态，是对基于人类创造力的真正发展，而不是纯粹的投机行为的强硬态度。

传统经济和实体土地之间的比例约为10：1。 如果与实体土地相比较，BSV将成为支持100万亿美元经济的 “土地”，而硬币（”土地”）的估值低于10万亿美元，或者10万亿美元的经济，而硬币的估值低于1万亿美元，如果它的规模较小。 然而，随着数字效率的提高，数字 “土地 “将比物理土地更有效率，所以数字版将有一个更好的比率。

请注意，在 “数字土地 “的情况下，上述更好的比例不赞成持有 “数字土地”，而赞成使用或开发它。 对于那些打算持有BSV的人来说，这可能感觉是一个不利的因素，但实际上，它将是相反的。 这是因为实际的经济价值不是一个人心中的主观想法，而是经济活动的结果。更快的发展似乎在任何时候都会把土地从土地所有者手中挤走，但如果你回过头来看，你会发现总是发展推高了土地价格。

相比之下，BTC已经接近1万亿美元的硬币估值，但支持的创造生产力的经济基本为零。这真是一个明显的反常现象。

BSV旨在吸引人们创造新的价值，而不是吸引希望在硬币本身上致富的搬运工。拥有BSV硬币的人可能仍然会像土地所有者那样得到回报，但这不是BSV经济的主要特征。

在BSV上，每个人都有机会通过为生态系统增加效用来创造价值。而且感谢上帝，人类的创造力并不是一种可以使用不公平的人造系统进行分配的人造资产。

事实上，BSV生态系统现在已经准备好了创造力的爆发，也许至少在未来几十年内都会持续下去。

更具体地说，根据最初的中本聪设想，比特币（BSV）旨在同时推动五个伟大的转变。

(1）货币的去中心化（通过硬币本身）

(2）资产的去中心化（通过代币化、脚本化和智能合约）

(3）数据的去中心化（通过Metanet和Teranode，见nChain）

(4）计算的去中心化（通过链上虚拟状态机；与分布式计算不同）

(5）人工智能的去中心化（上述#3和#4的组合）

上述所有这些都需要并承诺创造价值。

比特币SV是建立经济体的 “土地”

比特币SV不仅仅是一个经济体，它是一块巨大的 “土地”，已经被预先规划成2.1万亿个 “地段”（每个地段由一个satoshi代币代表）。 在每个地段（一个satoshi代币）上，可以创造一块经济（一个价值）。

这是通过可扩展的代币化协议来实现的，它使 “再代币化 “过程能够在一个或多个比特币代币（satoshis）之上创建一个不同的代币。 可以创建各种新的代币，符合法律规定的代币，兼容SPV的代币，多个用户的代币，可替换的代币，不可替换的代币，发行人许可的代币，等等。与智能合约相结合，这些代币将实现创新的商业模式，甚至支持新的经济。想象力是唯一的限制。

请注意，我们不是在谈论聪币本身的价值，而是在聪币之上创造的新价值。

一旦这种价值创造完成，底层的 “地段”（satoshi代币）就充当了价值的载体，因此，比特币（和每一个satoshi代币）所享有的所有运输功能，如安全性、防止双重消费、不变性、透明度、不可抵赖性、假名等，也被相关价值代币所享有。

同样重要的是，价值代币本身可以有一个独立的经济价值，由其自身的经济因素和市场决定，基本上与基础聪币的价值无关。 这就好比说，一个人可以在一块土地上建一栋楼，这栋楼可以有自己的价值，而不一定要从基础土地上得到，只不过，在比特币的情况下，”土地 “的价值和 “楼房 “的价值之间的这种分离比真正的土地和楼房要彻底得多。

这有巨大的影响。如果你认为比特币很贵，那就等着看那些建立在比特币土地上的更有价值的经济体吧。记住，当你买一个比特币时，你是在比特币土地上买了1亿个 “地块”。

很明显，BSV比特币区块链是数字价值网络新世界中的 “土地”。

与此相反，BTC想成为数字黄金。即使它真的成为数字黄金，它也不可能成为新数字世界的 “土地”。土地的价值比黄金要高得多，这一点是很容易体会到的，更不用说将建立在新 “土地 “上的整个 “房地产 “经济。

在某种程度上，BSV与BTC相比更像以太坊，只是在以太坊上，经济活动的成本比BSV高1000多倍，可扩展性也差1000倍。

但是，尽管以太坊的系统费用高得离谱，可扩展性低得令人痛苦，但以太坊却迸发出了创造力，这一点是值得关注的。它是通过允许人们做一些BTC故意从最初的比特币功能中砍掉的事情而实现的！它告诉你有多大的能量。这告诉你新技术有多大的能量。一旦有更合适的平台出现在更广阔的视野中，整个场面就会爆发出来。

黄金与土地

有人可能会说，如果BTC像 “数字黄金”，BSV像 “数字土地”，那么我宁愿买黄金也不买土地。

但 “数字黄金 “并不是一个事实，而只是为了诱惑人们而精心编造的一种叙述。 如果你相信这个叙述，而且它符合你的个人投资风格，没有人可以阻止你。

尽管如此，人们必须了解实物和数字之间的本质区别。

(1) 所谓的 “数字黄金 “可以很容易地被另一种数字黄金取代，而这种数字黄金恰好获得了优越的属性。 实物黄金则不然，这不仅是因为从经验上看，它有几千年的历史可以证明没有另一种贵金属可以与黄金竞争，而且更根本的是，作为一个原则问题，可以肯定的是，大自然只提供有限的选择，一种新的稳定和美丽的金属是不会莫名其妙地出现的。在数字世界中，情况则完全不同。

(2) 相比之下，一旦建筑物已经建在上面，土地就不容易被取代。 这不是一个技术问题，而是一个经济问题。

(3) BSV事实上也可以是 “数字黄金”，尽管这不属于它的叙述范围。 在实物中，黄金是黄金，土地是土地，土地不能也是黄金或成为黄金。 但在数字领域，”土地 “也可以作为黄金，反之亦然。 事实是，尽管BSV是为 “数字现金 “的交易能力以及 “数字土地 “的开发功能而设计的，但它也具有BTC所声称的 “数字黄金 “的属性。 与此相反，为了推销 “数字黄金 “的有意狭义叙述，BTC明确地将成为 “数字现金 “或 “数字土地 “的任何可能性作为一个原则问题排除在外。 对BTC来说，如果它不是黄金，就是垃圾，就是故事的结局。
