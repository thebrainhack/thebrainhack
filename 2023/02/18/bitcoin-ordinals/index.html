BTC上的NFT——它会持续下去吗？

本周，我们将关注一种被称为“ordinals”的上升趋势，这是我第一次在 Twitter 上偶然发现的。

比特币序数 (bitcoin-ordinals，简称BO) 是一种上升趋势，它提供了一种在比特币 (BTC) 区块链上构建不可替代代币 (NFT) 的方法，类似于收藏交易卡。比特币序数理论 (BOT) 由 Casey Rodarmor 创建，允许人们在 BTC 上发布 JPEG，这将 BTC 分为两个阵营：BTC Maxis 和 BTC Ordies。比特币 (BSV) 社交网络 Twetch 推出了一款名为 Ordinals 钱包的产品，以利用 Ordinals 热潮。Ordinals 热潮可能会迫使核心开发人员重新考虑他们的策略并增加 BTC 区块大小（即使不受欢迎）。

什么是ordinals(序数)？

序数就像比特币 (BTC) 区块链上的不可替代代币 (NFT)，类似于收藏交易卡。它们具有独特的编号、名称和附加资产，有些比其他的更稀有。人们出于各种原因收集它们，包括研究早期 NFT 的“NFT 考古学家”。

什么是比特币序数理论？

比特币序数理论（简称 BOT）是序数背后的理论。它由 Casey Rodarmor 创建，最初发布在他的博客上。简而言之，他提出了一种直接在 BTC 上构建 NFT 的方法，这种方法不需要“单独的代币、另一个区块链或对 BTC 进行任何更改”。

正如您所见，Ordinal 品牌已经吸引人们在此基础上进行建设。以上是以太坊名称服务（ENS）的 BTC 版本。

有什么好处？

简而言之？您现在可以在 BTC 上发布您的 JPEG（类似于NFT）。

这对比特币的发展有何影响？

在过去的十年中，由于块大小限制和 2017 年SegWit （隔离见证）的实施，BTC 几乎没有发生任何发展。有争议的升级意味着比特币的协议发生了变化，以允许诸如闪电网络和Taproot之类的东西并将大部分开发工作留给了 Bitcoin Core 和朋友们。

因此，多年来，大多数开发人员都转向了竞争区块链，例如 Ethereum、Solana、Polygon 等。

然而，如果不是 Taproot“升级”中的漏洞利用（漏洞），Ordinals 就不可能在 BTC 上实现。

有趣的是，BTC 现在又分为两个阵营：

  • BTC Maxis：“比特币仅用于 HODLing，因此 NFT 和 JPEG 是网络上的垃圾邮件”
  • BTC Ordies：“如果比特币可以做以太坊所做的，让我发布我的 JPEG”

所以现在有一大群“山寨币”开发者和 NFT 艺术家希望将他们的作品上传到 BTC，尽管这是对 Taproot 的无意利用。BTC Maxis 肯定在网上表达了他们的担忧：

这是第一次在比特币上尝试 NFT 吗？
不。比特币 SV（BSV）——一个试图将比特币恢复到其原始设计的项目——从一开始就有能力做到这一点。

如果有的话，BSV 的支持者向世界展示了比特币总是可以做所有其他区块链声称要做的事情（然后是一些）。但由于政治和个性，大多数人要么不知道，要么不愿听。

现在，BTC 的人们正在学习多年来已经教给 BSV 开发人员的东西。从长远来看，这可能是一件好事。

序数有任何风险吗？

很多。但其中许多并不明显。一些包括：

1、可能影响 BTC Core 开发者的重大诉讼。 这也可能影响多年来建立的任何网络效应。

2、未经许可上传的作品的版权问题。 任何区块链都只是一个分类帐——一个被设计成可追溯的）

3、内存池问题。 BTC 的区块大小上限为 ~4MB——这是 2017 年 Segwit 升级的遗留问题。因此，强制进入 BTC 网络的 NFT 越多，给它们带来的问题就越多。

下一步是什么？

比特币新发现的 NFT 炒作吸引了 BSV 开发者 Twetch 的兴趣。

作为所有行动的结果，Twetch的创建者推出了一款名为Ordinals Wallet 的新产品。

Twetch （地址：https://twetch.com）是一个比特币 (BSV) 社交网络，您可以在其中赚取比特币。

他们为 Ordinals 热潮做好了充分准备，因为他们多年来一直在 BSV 上使用 UTXO 进行构建，但他们一直是比特币本身的粉丝（无论股票代码如何）。

但 BTC 的用户可能很快就会明白为什么多年前有些人主张更大的区块大小：

因为如果比特币总是能够做 NFT，为什么它仅限于“数字黄金”和 HODLing？如果有足够的需求，这可能会迫使核心开发人员重新考虑他们的颠覆并增加块大小。但可能是一项太大的任务，并导致网络出现更多问题。

至于我？我的个人策略保持不变。坚持首要原则和长期建设。关注后台的区块链专利战。并且不要屈服于快速致富的计划或集体思考。

记住：通过耐心和坚持，它会来的。
