万物互联中的比特币和IPv6

比特币和 IPv6 实际上处于共生关系中，并将在未来几年看到互联网的边界被推向边缘。这是因为现代网络的许多方面已经开始应对 IPv4 标准下可用地址空间的迅速减少。

随着数十亿台设备连接到互联网，并且每天都有更多设备联网，IPv4 定义的 32 位地址空间意味着我们很快就用完了用于寻址单个设备的唯一地址。网络地址转换 (NAT) 暂时解决了这个问题，这是一种集中式服务修改途中数据包的 IP 标头以确保它们到达正确目的地的方法。

然而，与 IPv6 促进的对等通信相比，这是一种不必要的低效率。IPv6 是互联网协议的新版本使用户能够直接从其他用户的设备进行连接——在这种意义上，互联网上的每台设备都可以拥有自己的地址，以便使用对等连接进行通信。

在最近的演讲中埃克塞特 IEEE 活动，怀特说，这最终构成了什么是比特币的总体结构将被卷入其中，这正在将一切推向边缘。他补充说，边缘计算将事物从“云”变成了“雾”，其中有许多小端点和个人能够直接相互通信。

尽管节点记录交易，但它们不应该作为主要用于记录交易和处理的方法。节点对交易的顺序进行投票，并接受他们首先看到的交易。

怀特说，当比特币被创建时，进行交易的主要方式是通过 IP 到 IP 交换。比特币是一个直接的通信系统，一种使用户能够直接交换价值的方式。因此，系统反映了现金。

没有所谓的“私有”区块链

怀特还重申，不存在所谓的“私有”区块链。这是因为安全模型比特币不是共识方法而是哈希标头是（公开）分发的事实。

‘节点根本不应该是匿名的——但用户可以。这样做的目的是你可以有一些来来去去的实体来代替中介的位置，这样我们就可以进行小额支付。

‘比特币不是为了去中心化而去中心化. 它是一个旨在提供小额支付和安全令牌传输的系统。它使用分散式架构来实现这一目标。这意味着没有私有区块链，因为安全性在于将每个哈希标头发送给每个人。

权力下放不是一个政治术语

怀特说“权力下放”不是一个政治术语而是关于创造弹性。

‘假设世界上有五个节点，它们都是平等的，每人获得大约 20% 的奖励。然后一个被黑客干掉了，然后现在你有四个追逐同样多的钱。当你将这 5 个丢失的百分点添加到这些节点时，它们现在获得了总利润的 25%，这使其成为一个非常有吸引力的命题。

“突然之间，其他人会开始运行他们自己的节点，因为他们想在这个行业中竞争。不是因为您听说过的任何安全概念，而是因为经济学。有很多贪婪的商人喜欢赚钱，因此他们会建立企业来拿走你的钱。

比特币、IPv6 和边缘计算

Wright 补充说，去中心化的最终目标是边缘计算。他指出，以太坊等其他区块链注定会失败，因为它们依赖于在全球范围内运行相同的实例——这意味着你拥有的最好的是链中最薄弱的环节。

“如果你把它分开，让每个人都在边缘行动，那么你所做的就是验证并记录发生的事情。然后计算是网络上所有内容的总和。

为此原因，比特币协议在设计时始终考虑到 IPv6，因为在全球网络上单独寻址设备的能力是扩展区块链作为点对点互联网的支付和数据交易系统的效用的关键因素。

虽然节点记录交易，它们不应该作为主要用于记录交易的方法和处理。节点对交易的顺序进行投票，并接受他们首先看到的交易。

Wright 说，当比特币被创建时，进行交易的主要方式是通过 IP 到 IP 交换。比特币是一种直接通信系统，使用户能够直接交换价值。因此，系统反映了现金。
