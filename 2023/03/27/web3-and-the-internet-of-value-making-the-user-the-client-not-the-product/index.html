Web3和价值互联网：让用户成为客户，而不是产品

Web3 和价值互联网 (IoV) 是任何在区块链和数字货币行业工作的人无疑会听到的两个术语。在上一次投机性牛市中，它们成为流行语，让人联想到由虚拟现实 (VR)、人工智能 (AI) 和 去中心化网络提供支持的未来互联网。

虽然许多关于 Web3 和 IoV 的想法只不过是炒作和营销噱头，但这些概念本身是真实的，这种松散联系的想法混合将永远彻底改变互联网的运作 方式 。

那么，什么是 Web3？什么是车联网？他们会解决什么问题？大规模可扩展的 公共区块链将 在这一切中扮演什么角色？让我们从互联网的快速历史开始，让我们跟上速度。

Web1、Web2 和 Web3——互联网简史

Web1 是互联网的第一次迭代。它有时被描述为“只读”互联网。在这个时代，一小群人创建了基本网站，其中包含公众可以通过 Netscape Navigator 等早期网络浏览器阅读和浏览的有用信息。

在 Web1 时代，互联网的基础设施已经建成，最早的电子商务商店出现了，第一个让人们获取信息的工具也出现了。在那些日子里，大多数人认为互联网就像数字黄页一样，带有电子邮件作为附加组件。

然后是 Web2。这不是一夜之间发生的。这不像是有人按下开关，我们突然之间就进入了一个更复杂的互联网。这是一个缓慢、稳定但不懈的演变。

在 Web2 上，用户既可以读也可以写。数以百万计的博客在网上涌现，社交网络腾飞，最终，随着移动互联网的普及，应用程序成为一种东西。用户体验显着改善，但随着 Facebook（纳斯达克股票代码：META）、推特和谷歌（纳斯达克股票代码：GOOGL）等大型公司开始拥有网络，收集用户数据并将其货币化以获取利润，用户成为了产品。

Web2 是几年前 Web3 开始发展之前我们所处的位置。我们仍处于 Web2 时代，但随着区块链技术、数字货币、人工智能和其他下一代技术的快速发展，情况正在发生变化。

虽然 Web3 尚未大规模成为现实，但我们正在看到这个新互联网中的第一个应用程序正在形成。这个版本的互联网承诺是 点对点的，切断了中间人并结束了他们审查辩论和自由对话的能力。它还将允许用户在写入全球可用的可扩展公共区块链时拥有自己的数据，从而结束已成为 Web2 特征的数据收集。Web3 将由小额支付推动 ，使用户成为客户而不是产品。

目前，我们正处于 Web3 的风口浪尖，但还远未到那一步。Web2 正在消亡，人们受够了它并认识到它给社会带来的许多弊病。我们要完全采用 Web3 还需要一些时间，但在互联网的下一个十年里，人们非常乐观，希望我们能实现这一目标。

Web3 为企业和政府解决了哪些问题？区块链扮演什么角色？

Web3 通常被描述为互联网的读+写+自己的迭代。包括企业和政府在内的用户将真正拥有他们写入支持 Web3 的公共区块链的数据。与 Web2 不同，任何人上传到 Facebook 或 Twitter 的所有内容都归拥有数据库的公司所有，在这个时代，数据库将是公开的，数据所有者仍将是创建它的人。

Web3 和支撑它的无限可扩展的区块链将为企业、政府和用户解决几个问题。

数据主权

排在首位的是 数据主权。众所周知，数据收集已融入 Web2 的本质，而运行我们所依赖的服务的巨型垄断企业就喜欢这种设置。我们被认为是 Web2 中的产品。

然而，在 Web3 时代，创建数据并将其上传到区块链的公司、政府和个人将永远拥有它，并能够将其货币化。更何况，我们将成为客户，不再被视为产品。

有了数据主权，解决了几个问题，并创造了许多新的机会。 由于收集数据的窥探被搁置或完全消除，隐私得以恢复。大规模数据泄露将变得罕见，因为几乎没有集中式数据库可供攻击。包括企业和企业在内的用户将能够在适合他们的时候租用他们自己的数据以获取收入，而不是免费赠送。

许多人仍然无法理解具有真正数据主权的互联网是如何运作的，因为我们从一开始就不知道它会存在。有了区块链驱动的 Web3，我们会做到的。

透明度和可追溯性

与一些无政府主义者和被误导的基于区块链的数字货币的早期采用者的说法相反，Web3 将与他们设想的“去中心化”无法无天的世界背道而驰。事实上，在像 比特币 SV (BSV) 这样可大规模扩展的公共区块链的支持下，每笔交易的记录，无论是支付还是通信，都将在公共分类账上可见。

等一下！我刚才是不是说每一次通信和支付都会在 Web3 的区块链上可见？不，我没有。我说过 每笔交易的记录都是 可见的；这是一个巨大而重要的区别。虽然支付、通信和共享的数据将是绝对私密的，但发生此类交易的记录将在区块链上留下不可更改的时间戳。

这为政府和企业解决了几个问题。盗窃和腐败将更容易被发现并因此而受到抑制。被盗资产将被追回。可以快速定位任何文件或记录，并且使用正确的 私钥对其进行解密将变得简单而轻松。

在这个由区块链驱动的世界中，将有可能全面了解组织内部发生的事情，包括当前财务状况或谁上传或访问记录以及他们实时访问记录的时间。所有这一切都将彻底改变企业和政府组织的性质，使它们更加透明和负责任。

尽管 Web2 是一个试图从分散的数据库中拼凑出拼图碎片的时代，找出数据泄露发生的时间或资金转移到何处以及由谁转移，但在区块链驱动的 Web3 上，所有这些都将在几分钟内就可以知道。

点对点交流

Web2 的特征之一，或者可以说是区分它的东西，是大公司和所谓的受信任的第三方在一切的中间。

想一想；您的所有电子邮件都通过谷歌、微软（纳斯达克股票代码：MSFT）或雅虎拥有的服务器运行。您的私人聊天通过由 Meta、Twitter 或 Snapchat 控制的服务器进行。你最亲密的时刻，构成你生活的照片和视频，都保存在公司拥有的服务器上，这些公司将它们视为可以利用的商品。每笔 付款都 必须由银行或付款处理商清算。

虽然企业赚钱并没有错，但这种形式的监视资本主义却有些缺乏灵魂。事实上，它几乎不能被称为 资本主义 ，因为运行这些系统的垄断扼杀了竞争并扼杀了处于萌芽状态的竞争者。

在 Web3 的世界里，点对点的交流将成为常态。由于像比特币 SV 这样的区块链上的小额支付，将有可能支付节点来处理从发送者的设备直接移动到接收者的消息和电子邮件中的数据。有了 IPv6，无数的 IP 地址将变得可用，从而可以连接组织的所有设备，允许所有成员（无论是员工还是代理）通过加密的对等网络安全私密地进行通信。

当然，仍然需要构建促进这些点对点交换的应用程序。虽然已经有早期的迭代，但不可否认的是，Web3 还缺乏 Web2 提供的流畅的用户界面和体验。有远见的公司有机会利用这个机会并构建未来的应用程序，这些应用程序将为这些点对点交换提供动力，无论是支付还是电子邮件。

随着 Web3 的接管，我们所依赖的大公司将随着时间的推移变得越来越不重要。与其依赖于对通信和社交网络的垄断，他们还必须进行创新，开发人们想要使用的应用程序，最重要的是，他们必须在开放协议的基础上再次在市场上展开竞争。这将再次迎来自由市场，让每个企业都有公平的机会在公平的竞争环境中竞争，并培育出一个充满创新和机遇的新时代。

Web3 与我们今天所知道的完全不同

Web3 和 IoV 将与我们今天所知的互联网大不相同，以至于它们几乎不能被视为相同。要实现所有好处，大规模可扩展的公共区块链将需要发挥作用。
