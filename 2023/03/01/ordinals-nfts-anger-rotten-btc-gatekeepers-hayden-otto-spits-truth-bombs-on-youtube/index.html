Ordinals激怒BTC看门人——海登奥托在YouTube吐露真相

Ordinals NFT 似乎无处不在，席卷了 BTC。

在开发人员 Casey Rodarmor 利用 Segwit 和 Taproot 引入的更改后，BTC 区块链上写入了数万条铭文，引发了 BTC 核心开发人员和小区块极端分子的愤怒、屈尊和直接要求审查的呼声。

YouTuber Hayden Otto 在他的频道中大声疾呼那些自命为比特币看门人的极端主义者，并揭露他们行为中的矛盾和虚伪。

序数激发了人们对 BTC 的新兴趣

Otto 首先对Ordinals NFT 的情况进行了高级概述。他概述了由于 BTC 核心开发人员（如 Luke Dashjr）引入的协议更改，开发人员 Casey Rodarmor 如何找到一种在 BTC 区块链上启用 NFT 铭文的方法。

与以太坊 NFT 不同，Ordinals 被记录并存储在 BTC 区块链上。具有讽刺意味的是，即使在Vitalik Buterin和其他以太坊开发人员反复尝试解决陷入困境的区块链上离谱的汽油费之后，BTC 上的铭文比以太坊上的铭文要便宜得多。

序号导致一年中每天的交易量最高，据一些消息来源称，它们约占 BTC 交易的 50%（并且还在增长）。这为矿工带来了更多收入，并创造了 BTC 历史上最大的区块——高达 4MB。

尽管有这种积极的能量，关于比特币能力的新对话，以及矿工收入的增加，自封的 BTC 看门人对 Ordinals 表示厌恶和愤怒，称他们是对网络的攻击。

当他们自己攻击它时，他们正试图将序号描绘成一种攻击，——海登奥托

矛盾和虚伪比比皆是，但这并不是什么新鲜事

为什么 BTC 极简主义者对 Ordinals 如此愤怒？他们真的是认为 BTC 只应充当数字黄金的纯粹主义者吗？虽然他们可能会试图描绘那幅图画，但 Otto 指出，只要稍加挖掘，就很容易看出议程是什么。

Blockstream 是一家由Adam Back 、Lukedashjr 和 Gregory Maxwell创立的公司，它建立了营利性侧链来收取各种比特币交易的费用。虽然这些作为扩展解决方案出售，但实际上，它们是重新引入受信任的第三方，并且由于 Blockcstream 控制着它们，如果人们使用它们，它们可以收取无限的费用。

其中一个侧链（Liquid）恰好有一个与之竞争的 NFT 协议。Back、Dashjr 和最响亮的极简主义者会不会出于自身利益而非原则行事？人们不需要对人性思考太久就可以弄清楚这一点。

Back 呼吁矿工彻底审查 Ordinals 交易。在遭到强烈反对后，他撤回了自己的声明，并表示需要“教育”人们为什么序号对 BTC 不利。抵制审查制度怎么了，亚当？

除了 Back 之外，Luke Dashjr 一直是 Ordinals 最响亮的批评者之一。除了在 Twitter 上踢打和尖叫外，他还创建了一个补丁来帮助所谓的节点运营商忽略 Ordinals 交易。最终，这是徒劳的，因为任何关注过的人都知道，主节点什么都不做，而 Ordinals 交易由真正的节点（矿工）处理，他们非常乐意将费用收入囊中。

这是一位 BTC 极简主义者，正在接受真正的比特币支持者的教育，了解忽略 Ordinals 交易是徒劳的，以及他的“节点”实际上是多么无用。他甚至承认它什么也没做，只是说他选择忽略它们是最重要的。

在他富有洞察力的视频中，Otto 提出了一个相关且非常重要的问题：当 Lukedashjr 承认他最近多次遭到黑客攻击时，他如何仍然可以访问 BTC Core 的回购协议？毕竟，在 2016 年，安全漏洞被用作撤销 Gavin Andresen 访问权限的借口。难道 Andresen 关于中本聪身份的公开声明是他被踢出局的真正原因吗？无论如何，这是 BTC Core 开发者更加公然的虚伪。

所有这些都可以在比特币 SV 上做得更好

虽然 Casey Rodarmor 显然是一位聪明的开发人员，他应该受到赞扬，因为他唤醒了人们对 BTC 等甚至是破碎的比特币协议分叉的一些隐藏功能，但遗憾的是，他只见树木不见森林，并且仍然不了解比特币的真正含义能力。他最近在 Twitter 上对 BSV 发表了一些严厉的评论，尽管社区中的许多人都支持 Ordinals，并且尽管收到了一些 BTC 极端主义者的死亡威胁。

然而，当 BTC Core 独裁者不可避免地采取行动阻止 Ordinals 时，他和其他人有望意识到比特币作为 BSV 的真正力量。在 Satoshi 的原始协议中，欢迎和鼓励像 Ordinals 这样的开发，并且登记的平均费用只是成本的一小部分，没有缩放上限。

慢慢地，但最终，至少一些 BTC 拥护者可能会觉醒并开始提出问题。虽然 Rodarmor 本人对 BTC 过于执着，无法完全理解，但成千上万的其他人正在观看、疑惑并开始质疑这种说法。

BTC 社区绝对是狂野的。
