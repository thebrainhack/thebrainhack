使用GorillaPool设定挖矿新标准

在最近于奥斯汀举行的 Unbounded Perspectives 活动中， Gorilla Pool的创始人兼首席执行官Kurt Wuckert Jr介绍了他的公司作为独家比特币 SV 矿池的服务和运营。该演示文稿包括ARC、Metanet 和 Gorilla Pool 的一款名为 Jungle Bus 的最新产品。

在开始他的演讲之前，Wuckert Jr 简要概述了比特币挖矿的过程。交易被广播到网络并由节点收集，节点是收集数据的计算机。

使用ASIC 矿工，个人竞争寻找有效的工作证明并构建区块。然后将该块传播并提议到网络。如果被接受，矿工将收到补贴中的代币以及与该区块相关的费用。

Wuckert Jr 随后解释说，Gorilla Pool 运营着巨大的数据服务器（节点），这些服务器作为其矿池的基础设施。该池允许集体哈希算力一起解决问题，并根据贡献的哈希量分配支出。这包括提供托管服务和移动数据中心。
https://youtu.be/y3xP3HKDn7k

Wuckert Jr 强调了 Gorilla Pool 的 ASIC 计算机的效率，并强调了对哈希基础设施和经典服务器基础设施的需求，以验证交易和传播区块。他注意到运行比特币节点所需的专业化，并鼓励将此类操作外包给他们的公司。

他提到了他们在亚洲国家的 ASIC 制造和不同地点的采矿业务的全球影响力，包括魁北克、华盛顿州、北卡罗来纳州、迈阿密、德克萨斯和欧洲。他们运营的分布式特性确保了对政治、地理或天气相关因素的弹性。

Jungle Bus——建立在 BSV 上的公司不再需要运行全节点

Wuckert Jr. 强调了寻找真正精通比特币开采和围绕比特币构建基础设施的公司的重要性。他批评各种比特币挖矿公司，称其中许多公司都是假的，并没有在挖比特币方面盈利。然后，他讨论了比特币网络带来的挑战以及对可靠且具有成本效益的数据访问的需求，以及 Gorilla Pool 旨在在这方面实现这一目标的新产品——Jungle Bus。

Jungle Bus 旨在提供对区块链数据的逻辑和高效访问，并为不同的协议定义逻辑，并以记录的方式提供分散的数据访问。他解释说，Jungle Bus 既简化了业务又降低了业务成本：

“不是以每月一千美元的成本运行一个完整的 BSV 节点，再加上拥有一个系统管理员所需的一切，我们确保你的数据一直在运行并且数据是好的。”

‘那已经是很多钱了。你可以每月向我们支付大约 15 美元来同步你需要的数据，并将其保存在你自己的系统上，并拥有对我们节点的 API 访问权限，”他说。

这种方法消除了运行完整节点的需要，并提供具有信誉、保证和潜在保险的可靠数据访问。Wuckert Jr 解释说，Jungle Bus 是目前没有竞争对手提供的矿池中独一无二的产品。它是在业务运营中有效利用区块链数据的专有解决方案。

Metanet 和 ARC——成为比特币的 ISP

Wuckert Jr 解释了Metanet 的概念，Metanet 是比特币 SV 之上的覆盖网络，其功能类似于互联网。该网络允许通过比特币交易加密和传输数据包，提供增强的安全性和更快的数据传输潜力。

他将他们的愿景比作比特币的 ISP（互联网服务提供商）。他描述说，未来用户将能够以类似于 AT&T、Verizon 和 Comcast 等公司提供的高速互联网访问的方式使用比特币。

然而，Wuckert Jr 提到，现有解决方案目前存在挑战和局限性。为了解决这个问题，Gorilla Pool 测试了比特币服务 ARC，这是一种高性能交易引擎，旨在比现有解决方案更高效。ARC 有一个内置的 tumbler，可以将交易重新广播到他们的节点和比特币网络的其余部分，确保数据完整性并减轻潜在问题。

了解更多关于 ARC 的信息——尖端的交易处理器，它简化了向比特币网络提交交易的过程。通过提供无缝连接和卓越的性能，ARC 为 BSV 生态系统中无与伦比的全球可扩展性铺平了道路。
https://www.bitcoinsv.com/arc

Wuckert Jr 指出，该产品尚未被其他公司广泛部署或理解，正在与比特币协会合作开发，最终将开源。他补充说，Gorilla Pool 是目前唯一能够有效运行该产品的产品。

Wuckert Jr 在结束演讲时指出，Gorilla Pool 在构建应用程序堆栈、建议合适的钱包和协议以及提供可靠的客户支持方面提供帮助。他强调他们致力于展示比特币的真正潜力，并邀请任何有兴趣与他们合作的人随时与他们联系。
