比特币代币战争2.0？

比特币，一种电子现金系统，不仅仅是一种货币或一种新的货币形式，而且还是一个技术平台，直到最近几个月，很少有人理解这一点。最近，BTC 上的 Ordinals 协议引发了数字代币的狂热，这些代币人气暴涨，并在“加密”市场停滞和股票及其他传统资产的主流市场价格走势普遍持平的情况下引发了代币牛市。

但是你知道这不是第一次比特币代币繁荣吗？事实上，这甚至不是第一个比特币数据时代。这是正确的！我们以前来过这里。

中本聪以时报头条的“任意”数据拉开了区块链的序幕，证明代币没有预挖。一个完美的时间戳证明，让世界永远记住。但是今天，许多拥护者会称这样的事情为垃圾邮件。

在突发新闻中，已知的第一个使用比特币进行的购买可能是通过与中本聪本人参与的论坛讨论获得的屏幕保护程序大小的 JPEG。增加了对比特币作为现金的有用性以及数字艺术可能比比特币本身更有价值的奇怪观念的可信度。在这种情况下，似乎有人对这个屏保的估价超过了五百个比特币！

多年后，当我们进入关于比特币、电子现金和代币价值的新一轮辩论时，重要的是要记住，对于代币拥护者来说，它从来没有结束，而且比特币似乎一直在投资大师式的倡导下失利.

或许这一次，不一样了。

语境

很少有人知道，比特币的底层技术只是一个分布式数据库、强大的脚本语言和一个通信网络，它们结合在一起可以赚取稀缺的可编程货币。比特币最常见的用途是使用解锁脚本发送的简单支付，但在使用脚本堆栈方面更具创造力，开发人员已成功创建代币或独特的数字资产，可以代表一件艺术品中的任何东西，证明购买或类似房地产的东西。

很少有人知道这个概念几乎和比特币本身一样古老！

彩色硬币和交易对手代币（两种形式的比特币代币）的出现在比特币社区内引发了重大分歧，导致了通常所说的“比特币内战”。

比特币代币的起源

比特币代币的构想早在中本聪本人讨论自动售货机和成人网站中的代币时就已构想出来，但当时只是理论上的。2013 年，彩色币被设想用于在简单货币交易之外的交易中实施 SATS。今天的序号是基于这个前提的。
https://youtu.be/fmFjmvwPGKU

这个想法是将额外的信息嵌入到 比特币交易中，将它们变成代表任何形式价值的“代币”——而不仅仅是货币。这开辟了一个充满可能性的世界，从代币化资产到去中心化应用程序。

CounterParty代币的介绍

2014-15 年，Counterparty 代币将比特币代币的概念更进一步。彩色代币可以为特定的比特币“着色”，将其标记为代表某种资产或合约。另一方面，交易对手令牌允许使用 OP_Return 中添加的脚本在比特币区块链中创建自定义令牌，这允许更复杂的应用程序和金融工具。

当时，CounterParty 背后的公司正在招揽潜在客户，包括普通纸牌游戏和有兴趣在比特币上发行资产的大型零售商。

比特币内战：起因和战斗

然而，这些新的发展并没有受到所有人的欢迎。一些小型区块链纯粹主义者认为，这些代币损害了比特币作为一种简单、去中心化货币的唯一目的。其他人担心这些令牌的可扩展性和安全性影响。这些分歧演变成一场全面的冲突，辩论、分叉甚至人身攻击标志着 2015-16 年时代成为“比特币内战”的开端。

BTC 的著名核心开发人员 Luke Dashjr 发布了一个“补丁”，将 OP_Return 的大小减小到 40kb——通过一次全网软件更新就杀死了 CounterParty 的用例，并显示了比特币软件发行过程中的中心化问题。

后果

这场冲突的后果是显着的。这个时代见证了重大分裂的开始，社区分裂成不同的派系，每个派系都支持比特币的不同技术和哲学路径。这包括分裂社会群体、品牌“站队”并最终导致 比特币在 2017 年出现分裂。从那时起，比特币的价值和声誉就受到了影响——自 2017 年以来，作为统一的梦想，比特币失去了大约一半的市场主导地位，全球比特币货币已经贬值，取而代之的是一种基本上是胡说八道的拉高出货经济。

复兴

在 2017-2022 时代，大区块商（BCH，后来的 BSV）专注于代币标准、可扩展性和在比特币堆栈中本地交互的智能合约的研发。这包括 CounterParty 风格的 OP_Return 代币、像STAS这样的脚本风格的代币 ，以及一些混合系统，这些系统已经证明自己可以扩展到每个区块少于 300 万个代币交易——这个数字在任何其他区块链上都是不可想象的。

在小阻塞方面，BTC 收紧了网络的泳道以减少用例，并专注于限制带宽以鼓励节点数量增长，作为“去中心化”和“价值存储”的指标。

不过，最近，在一位名叫 Casey Rodarmor 的 BTC 开发人员创建了一个简单、优雅的代币化标准之后，这些旧辩论又重新浮出水面，该标准使用彩色代币方法的一种变体将不可替代的资产铭刻在 satoshis 上。由于它使用 sats 的序数计数方法，该协议被称为“序数”，它引发了关于交易有效性、垃圾邮件和其他出于政治动机的戏剧性争论的风暴。在最初的内战期间，大量资金投入到有影响力的人、媒体和社会活动中，以教导人们比特币不具备代币或可扩展性。BTC 上的序号铭文严重破坏了像 Luke Dashjr 这样的小区块贵族的权威主张，他一直在寻找一种方法来再次结束 BTC 上的这种工具。

随着比特币不断发展并发现其技术的新应用，围绕其目的和方向的同样的斗争正在重新进行。这种复兴是由与最初的比特币内战相同的热情和分歧推动的。它为新来者创造了一个机会，让他们可以重新审视促成这一切的讨论。BSV 派系最近发布了 1Sat Ordinals 协议，该协议利用基本的 Ordinal 代币化系统，但允许真正的单卫星交易、小额支付和分美分费用，使企业家能够专注于其代币的价值，而不是被被迫根据使用成本对它们进行估价。

类似的项目正在莱特币和狗狗币上涌现，因为它们与比特币分叉的基本相似之处。

期待

比特币及其代币的未来仍然不确定。社区会从过去吸取教训并找到调和分歧的方法，还是另一场内战不可避免？

BTC会分裂成有序链和无序链吗？自动收报机和哈希算力将如何摆脱困境？这会不会告诉人们BTC交易是不可变动的，让开发商贵族把自己画成一个叙事角？移除的威胁能否像 BTC 的主导地位在近六年的市场主导地位下降中一样，将更多的能量和发展推向其他链？

无论结果如何，比特币代币从一个新颖的想法到一个有争议的问题的旅程一直是过去十年比特币社区内更广泛斗争的迷人反映。
