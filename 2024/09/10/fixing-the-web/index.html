修复网络

我们该如何修复万维网？我对这个半固定的话题颇有研究，但最近我在专注于一些常规研究时发现了一个引人入胜的提议。

由蒂姆-伯纳斯-李（Tim Berners-Lee）领导的Solid 项目旨在彻底改变互联网管理和控制个人数据的方式。伯纳斯-李是一位富有远见的计算机科学家，他发明了万维网，他推出的 Solid 是一个重新构想数字隐私和数据所有权的框架。Solid 试图赋予个人权力，允许他们将数据存储在自己控制的分散式 “pod “中，而不是交给 Facebook（NASDAQ: META）、Google（NASDAQ: GOOGL）或亚马逊（NASDAQ: AMZN）等集中式平台。通过这样做，Solid 提供了一条通往更公平互联网的道路，在这条道路上，用户保留了对其个人信息的主权，可以决定谁可以访问他们的数据以及访问的目的。这种方法有望打破大科技公司目前的商业模式，这些公司在未经用户同意的情况下，将个人数据货币化。

听起来很高尚，而且与我自己的一些想法很相似，令人耳目一新！

Solid 的核心使命是解决网络上出现的根本性权力失衡问题–伯纳斯-李最初设计的网络空间是开放和自由的，但后来却被与监控国家、硅谷和一群大多数人难以理解的技术官僚实体关系密切的垄断公司所控制。

Solid 的架构围绕可互操作的数据舱展开，用户可以在其中管理自己的所有信息–从社交媒体互动到医疗记录–而无需不断将其上传到不同的集中式服务中。这种去中心化不仅为用户提供了更大的隐私和控制权，而且还引入了一个创新的时代，开发人员无需囤积数据即可构建与数据交互的应用程序，从而为一个全新的、以用户为中心的网络生态系统铺平了道路。

让我们静下心来，感谢蒂姆-伯纳斯-李对现代社会做出的无与伦比的贡献。作为万维网的发明者，伯纳斯-李的工作彻底改变了人类在全球范围内共享信息、交流和协作的方式。他一直坚定不移地致力于使网络成为一个开放的平台，即使他眼看着网络最初的承诺在企业贪婪和侵犯隐私的重压下被侵蚀。现在，通过 Solid，伯纳斯-李再次以领导者的身份挺身而出，推动建立一个尊重用户自主权和尊严的网络。他的愿景挑战着我们所有人，让我们重新思考互联网的未来–技术服务于人类的未来，而不是相反。

《坚实计划》既是对我们行动的呼吁，也是为重新找回我们的数字生活而提出的蓝图。它提醒我们，网络的基本原则–开放、协作和信任–不是过去的遗物，而是值得为之奋斗的理想。

但我们能做得更好吗？

我想是的，这对我来说非常重要，我感到天赐的挑战，让我接触到源头，确保我们利用这个机会解决一些根本性的问题。

所以，请原谅我在这里转换了语气。

致蒂姆-伯纳斯-李的公开信

亲爱的蒂姆-伯纳斯-李先生

不可否认，您对 Solid Project 的愿景植根于崇高的初衷–在一个隐私日益受到威胁的时代，让人们对其个人数据拥有更多的控制权。您对一个开放的、以用户为中心的互联网的承诺始终是明确的。然而，作为开放协议和高效数字系统的倡导者，我必须对 Solid 的做法表示担忧。

Solid 虽然概念上合理，但可能过于复杂，在实践中会对普通用户产生反作用。

Solid 的核心前提是让个人有能力管理自己的数据，但现实情况是，这一框架所要求的技术复杂程度是大多数人根本不具备的。设置、维护和安全管理数据 pod 所涉及的复杂性使得 Solid 实际上无法为普通人所用。这其中的讽刺意味不言而喻：在试图给予用户更多控制权的同时，这一过程却使他们更有可能误操作或忽视自己的数据，从而导致更大的漏洞和滥用。恕我直言，当一个系统如此难以使用时，它并没有实现控制的民主化，而是模糊了控制。

此外，Solid 在处理数据所有权方面存在重大缺陷。该系统缺乏一个强大的机制来验证谁拥有什么，这就为滥用版权打开了方便之门。从理论上讲，有人可以轻易地复制别人的文件，稍作改动，就宣称是自己的作品。如果没有强有力的、不可更改的所有权证明，Solid 就会在不经意间将用户暴露在潜在的知识产权盗窃和操纵的世界中。在数字内容就是货币的时代，这种疏忽会使系统变得不可靠，并破坏其基本宗旨。

数据就是金钱”，这是我的一个公理，因此，我有一些想法供大家参考。

这些解决方案以BSV区块链的形式存在，它利用一套协议和工具（如Metanet、BAP（比特币证明协议）、AIP（作者身份识别协议）和Sigma身份协议），利用比特币的私钥和公共证明功能，在公共场合展示所有权和出处。在 BSV 成为货币并将所有数据货币化的金融能力之上，这个以社交和网络为重点的框架允许创建、转移和管理非金融数字资产，其清晰、不可更改的所有权记录直接存储在区块链上。与所有权可能存在争议的 Solid 不同，BSV 区块链提供了一种加密安全的方式，通过不可逆转的公开时间戳来验证来源和真正的所有权，从而防止 Solid 容易被滥用的情况。

此外，BSV 的方法既简单又安全。终端用户的易用性是其核心特征，而不是事后才想到的。有了 BSV，交易（包括 NFT、代币和其他数字资产的创建和管理）都是无缝的，而且无论交易量大小，都只需几分钱。区块链的公共分类账确保了透明度和问责制，为数据所有权和数字资产管理提供了一个直接、可扩展的解决方案。BSV 交易效率高、成本低，是日常使用的实用工具。

虽然 “团结 “作为一个理论框架非常有效，并有很多伟大的核心理念，但其复杂性以及缺乏对数据和版权的适当保护，使其在现实世界中不那么安全和实用。

但我并不是要你放弃这个想法。相反，我建议采用一种综合方法，将 BSV 区块链作为底层工具，使 Solid 真正……稳固！

像 BSV 这样的公共区块链通过提供一个易于使用和维护的去中心化、防篡改系统，正面解决了这些问题。它的建立正是为了应对 Solid 所要解决的挑战，但它的实现方式却是易于访问、用户友好且本质上更加安全。

这是一个安全、开放的公共数据库，数据可选择加密/私有或公开，并可轻松地打上不变的时间戳！

让我们把这些功能添加到 Solid 协议中。

伯纳斯-李先生，您一直是互联网发展的指路明灯，并极大地影响了我对这些问题的思考！您对开放、公正的网络的承诺是无与伦比的。我相信，探索 BSV 的能力与您的目标是一致的，即利用 Solid 和 BSV 创建一个更公平、更安全的数字世界，开创我们都希望的未来。

我希望您能仔细考虑一下 BSV 如何与您的愿景相辅相成，并继承您的传统，为所有人开创一个更好的互联网。我很荣幸能与您直接联系，共同探讨如何让这一切取得巨大成功！

谨启
小库尔特-伍克特
